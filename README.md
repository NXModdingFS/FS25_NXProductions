<ins>**NX Productions**</ins> (ModHub v1.0.0.0 - Developement v1.0.0.2)

A production management mod for Farming Simulator 25 that provides an overview of all your production facilities with automatic management capabilities.

<ins>**Key Features**</ins>

- Production Overview: View all productions with fill levels, capacities, monthly income and processing type.

- Toggle Views: Switch between inputs, outputs, and recipes

- Financial Tracking: See monthly revenue, costs, and net profit for each production

- Auto-Management: Automatically activate/deactivate productions based on input availability and output capacity

- CSV Export: Export all production data for external analysis

- Logistics Page (v1.0.0.1) - Overview of all production's and their recipies includes, Supply Duration, Cycles/MO, Output Mode, Location, Status

- Multi-Language Support: Available in 12 languages

- Processing Type: See if the production supports Serial (S) or Parallel (P) processing. When the production supports Parallel processing, for every tick the server does, every active recipe advances. When the production only supports serial processing, for every tick the server does, only ONE recipe advances in turn.

**Usage:**

Press Right Shift + P to open the production overview.

**Settings Menu** 

Access settings via ESC → Settings → Game Settings → NX Productions

<ins>**Automatic Productions:**</ins>

**Automatically manages your production facilities:**

- Enabled: Productions activate when inputs are available and deactivate when outputs reach 95% capacity
  
- Disabled: Manual control only (default)

This setting checks run every 5 seconds and only affect productions with configured recipes. Changes are synchronized across all players in multiplayer.
